# 🍽️ Veloura – Luxury Fine Dining Restaurant Website

Welcome to **Veloura**, a modern and elegant luxury restaurant website designed to deliver a premium digital dining experience. Built with HTML, CSS, and JavaScript, this project focuses on sophisticated UI, smooth interactions, and responsive design.

## 🌐 Live Demo

🔗 **https://kas232.github.io/veloura/**

---

## ✨ Features

- 🎨 Luxury and modern UI design
- 📱 Fully responsive layout
- 🍽️ Elegant restaurant landing page
- 🖼️ Beautiful food gallery
- 📖 About section
- 📋 Signature menu showcase
- 📞 Contact section
- ⚡ Smooth scrolling and animations
- 💻 Clean and organized code structure

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
